- **Behavior**: Clicking overlay closes dialog

- **Border**: Rounded corners (8px)

## Responsive Breakpoints

### Mobile (< 768px)
- Dialog width: 95% of viewport
- Reduced padding
- Stacked footer buttons on small screens
- Touch-friendly button sizes (44px min height)

### Tablet (768px - 1024px)
- Dialog width: 80% of viewport
- Standard layout maintained

### Desktop (> 1024px)
- Dialog width: Fixed max 600px
- Hover states more prominent
- Larger interface item cards

### Dialog Appearance
- Fade in overlay: 200ms ease-out
- Scale in dialog: 250ms ease-out transform

### Loading States
- Spinner rotation: 1s linear infinite
- Button state transitions: 150ms ease-in-out

- Border color transition: 200ms ease-in-out
- Background color transition: 200ms ease-in-out

## Layout Specifications

### Spacing
- Dialog padding: 20px
- Section margins: 20px vertical
- Interface item spacing: 10px between items
- Button spacing: 10px gap

- Dialog title: 1.25rem, bold
- Interface names: 1rem, semi-bold
- Interface details: 0.85rem, regular
- Error messages: 0.9rem, medium
